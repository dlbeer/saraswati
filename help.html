Saraswati: text-mode EPUB reader

To get started, press o to open a file. Press Tab for filename completion. Use PgUp, PgDown and the cursor keys to navigate. Press h to make this help text appear.

Within any entry box, you can use the Up and Down keys to select an item from the history. When opening a previously read file, the cursor will skip to the last known cursor position.

Key reference

Ctrl+G          Cancel the current operation.
Ctrl+X          Quit.
\               Toggle half-width wrapping.
Home, <         Go to the beginning of the book.
End, >          Go to the end of the book.
Up, j           Go up a line.
Down, l         Go down a line.
PgUp, Ctrl+P    Go up a page.
PgDown, Ctrl+N  Go down a page.
Left            Go to the previous link in the line.
Right           Go to the next link in the line.
/               Search forwards.
?               Search backwards.
n               Find next.
p               Find previous.
Enter           Follow link.
Ctrl+H          Go back.
Ctrl+F          Go forwards.
#               Show link URL.
h               Display help text.
t               Show full table of contents.
T               Show short table of contents.
b               Go to book text.
o               Open file.
